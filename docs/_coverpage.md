# lib-datatable <small>(v0.0.11)</small>

<hr>

> A NodeJS library to manipulate an in memory table (array of arrays)

> Use Excel-like Addressing (A1)

<hr>

- Simple and lightweight (less than 12kb)
- No dependencies - 100% JavaScript
